PRODUCTS API

Get All Products

Endpoint:
GET /products

Description:
Retrieve a list of all available products.

Request Example (JavaScript):
fetch('https://fakestoreapi.com/products
')
.then(res => res.json())
.then(data => console.log(data));

Success Response:
HTTP 200 OK

Response Body:
[
{
"id": 0,
"title": "string",
"price": 0.1,
"description": "string",
"category": "string",
"image": "http://example.com
"
}
]

Add a New Product

Endpoint:
POST /products

Description:
Create a new product.

Request Body (JSON):
{
"id": 0,
"title": "string",
"price": 0.1,
"description": "string",
"category": "string",
"image": "http://example.com
"
}

Field Details:
id : integer (required)
title : string (required)
price : float (required)
description : string (required)
category : string (required)
image : string (URL, required)

Success Response:
HTTP 201 Created

Error Response:
HTTP 400 Bad Request

Delete Product

Endpoint:
DELETE /products/{id}

Description:
Delete a product by ID.

Path Parameter:
